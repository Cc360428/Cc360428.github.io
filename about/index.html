关于本博客

欢迎来到我的个人技术博客！这里是我分享自己在 Golang、Java 编程方面的经验和知识的地方。作为一名 5 年的程序员，我希望通过写作来记录和分享一些我的工作经验和学习心得，帮助其他开发者更好地掌握这项技能

博客内容

Golang

  • Base：介绍 Go 语言的基础知识和常用语法特性
  • Web：介绍如何使用 Gin、Echo 等 Web 框架进行开发，并讲解如何进行 RESTful API 设计和开发
  • 网络知识：深入理解网络模型、TCP/IP、HTTP/HTTPS 等协议，并介绍如何进行网络编程

Java

  • Base：介绍 Java 语言的基础知识和常用语法特性
  • Web：介绍如何使用 Spring Boot、Spring Cloud 等 Web 框架进行开发，以及如何使用 MyBatis、Hibernate 等 ORM 框架操作数据库
  • 算法：介绍 Java 中常用的算法，如排序算法、查找算法等

Python

  • Base：介绍 Python 语言的基础知识和常用语法特性
  • Client：介绍如何使用 Python 进行网络编程、HTTP 请求等
  • 开发工具：探讨常用的 IDE 和编辑器，如 PyCharm、VS Code 等

前端

  • HTML/CSS/JS/jQuery：介绍前端开发中最基础的技术，包括 HTML 标签、CSS 样式和 JavaScript 脚本等
  • Vue.js：介绍如何使用 Vue.js 进行前端开发，并讲解其组件化设计思想和生命周期钩子等
  • Bootstrap：介绍如何使用 Bootstrap 快速搭建响应式网页

系统

  • Linux：介绍 Linux 操作系统的基础知识和常用命令，并讲解如何使用 Linux 进行开发和部署
  • Windows：介绍 Windows 操作系统的基础知识和常用命令，并讲解如何使用 Windows 进行开发和部署
  • Mac：介绍 macOS 操作系统的基础知识和常用命令，并讲解如何使用 macOS 进行开发和部署
  • Utils：介绍一些实用工具，如文本编辑器、调试器、压缩软件等

数据库

  • MySQL：介绍如何使用 MySQL 进行关系型数据库设计和开发
  • Redis：介绍如何使用 Redis 进行缓存和分布式锁的设计和开发
  • MongoDB：介绍如何使用 MongoDB 进行 NoSQL 数据库设计和开发

工具和技术

  • Docker：Docker 进行容器化部署
  • Kubernetes：Kubernetes 使用 15%
  • Git： Git 进行版本控制，并讲解常用的 Git 命令和工作流程
  • 消息队列：RabbitMQ、Kafka 等消息队列进行异步通信和任务调度等
