Go 基础

初步认识 Go

  • Go 跨平台（Linux、Windows、macOS），同一份代码可通过 交叉编译 生成不同平台的二进制
  • 语法简洁、并发模型清晰，支持通过 CGO 调用 C 代码
  • 应用广（Web、基础设施、云原生、数据处理等）

Go 环境

Linux & macOS

  1. 下载 Go 安装包：在 Go 官方网站下载适合自己系统和架构的安装包
  2. 解压安装包：将下载的安装包解压到系统的某个目录下，如 /usr/local。
1

   sudo tar -C /usr/local -xzf go*.tar.gz
  1. 配置环境变量：编辑 ~/.bashrc 或 ~/.profile，添加以下内容：
1

   export PATH=$PATH:/usr/local/go/bin
  1. 使环境变量生效：执行以下命令使环境变量生效
1

   source ~/.bashrc

或者

1

   source ~/.profile
  1. 检查安装：打开终端，执行 go version，如果输出 Go 版本号则安装成功

注意：不同的 Linux 发行版可能有所不同，以上是一个基本的安装步骤，具体步骤可以根据自己的情况进行调整

Windows

  1. 下载 Go 安装包：在 Go 官方网站 官方网站下载适合自己 Windows 版本和架构的安装包
  2. 运行安装程序：运行下载的 Golang 安装包，按照提示进行安装，通常只需要选择安装路径和确认安装即可
  3. 配置环境变量：将 Go 的 bin 目录添加到系统的 PATH 环境变量中，这样才能在命令行中使用 Go 命令
  • 打开”控制面板” -> “系统和安全” -> “系统” -> “高级系统设置” -> “环境变量”
  • 在”系统变量”中找到”Path”变量，点击”编辑”
  • 在编辑系统变量窗口中，点击”新建”，输入 Golang 的 bin 目录，例如 C:\Go\bin
  • 点击”确定”保存修改。
  1. 检查安装：打开终端执行 go version，如果输出 Go 版本号则安装成功

注意：在 Windows 环境下，如果出现权限问题，需要使用管理员权限运行命令行终端

分隔符

一行代表一个结束，不需要像某些语言那样写 ;。Go 编译器会在合适的位置自动插入分号

标识符

标识符用来命名变量、常量、实体

  • 不能以数字开头
  • Go 语言关键字不能使用
  • 运算符不能使用

关键字

  • 25 个关键字：
    • break, default, func, interface, select
    • case, defer, go, map, struct
    • chan, else, goto, package, switch
    • const, fallthrough, if, range, type
    • continue, for, import, return, var
  • 预定义标识符（节选）：
    • 基本类型：bool byte rune string int uint uintptr float32 float64 complex64 complex128 error
    • 常量：true false iota nil
    • 内置函数：append cap close copy delete len make new panic print println real recover

Go Modules 快速开始

从 Go 1.16 起，默认启用 Modules。建议新项目都使用 Modules。

1
2
3

go mod init example.com/hello
go mod tidy
go run .

常用命令：

  • go env：查看环境变量
  • go list ./...：列出当前模块内所有包
  • go test ./...：运行所有测试
  • go fmt ./...：格式化代码

常见

  • 变量的声明必须使用空格隔开
  • 运算符之间加空格，适当加空格更好看
  • 未使用的变量会编译不通过
  • 写完代码之后常见使用 go fmt ./... 格式化代码

  • public And private

    不管变量名或者方法名，首字母大写就是 public，首字母小写就是 private

基础概括

  1. 语言特性和语法：Go 是一门静态类型、编译型、并发型、带垃圾回收的编程语言。
  2. 变量和常量：变量必须显式声明，可用 var 或 :=；常量使用 const。
  3. 基本数据类型：整型、浮点型、布尔型、字符串等。
  4. 控制流程语句：if、switch、for、range、break、continue、goto。
  5. 数组、切片和映射：数组是固定长度；切片可变长度；映射是键值对结构。
  6. 函数和方法：函数可独立调用，方法绑定到类型。
  7. 结构体和接口：结构体定义数据，接口定义行为。
  8. 并发编程：Go 内置 goroutine 和 channel。
