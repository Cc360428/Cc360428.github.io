Golang 基础

初步认识 Golang

Go 语言的特点

  • 跨平台编译: Go 支持 Linux、Windows、macOS 等多个平台，通过交叉编译可以轻松生成不同平台的可执行文件
  • 简洁高效: 语法简单清晰，学习曲线平缓，关键字仅 25 个
  • 原生并发: 内置 goroutine 和 channel，天生支持高并发编程
  • 自动垃圾回收: 内置 GC 机制，无需手动管理内存
  • 静态类型: 编译时类型检查，提高代码安全性
  • 快速编译: 编译速度快，接近解释型语言的体验
  • CGO 支持: 可以调用 C 语言代码，扩展性强

Go 的应用领域

  • Web 开发: Gin、Echo、Beego 等框架广泛用于后端服务开发
  • 微服务: gRPC、Go Micro 等支持构建分布式微服务架构
  • 云原生: Docker、Kubernetes、Prometheus 等云原生工具均用 Go 编写
  • 区块链: Hyperledger Fabric、Ethereum 等区块链项目
  • 大数据: 分布式数据处理、日志收集系统
  • 运维工具: Terraform、Consul、Etcd 等 DevOps 工具
  • 网络编程: 高性能代理、负载均衡器、消息队列

Go 环境

Linux & MacOs

  1. 下载 Golang 安装包：在 Golang 官方网站上下载适合自己 Linux 发行版和架构的 Golang 安装包

  2. 解压安装包：将下载的安装包解压到系统的某个目录下，如 /usr/local。

    1

    sudo tar -C /usr/local -xzf go*.tar.gz

  3. 配置环境变量：编辑 ~/.bashrc 或者 ~/.profile 文件，添加以下内容：

    1

    export PATH=$PATH:/usr/local/go/bin

  4. 使环境变量生效：执行以下命令使环境变量生效

    1

    source ~/.bashrc

    或者

    1

    source ~/.profile
  5. 检查安装：打开命令行终端，输入 go version 命令，如果输出 Golang 的版本号，则说明 Golang 安装成功

注意：不同的 Linux 发行版可能有所不同，以上是一个基本的安装步骤，具体步骤可以根据自己的情况进行调整

Windows 安装详解

方法一：安装包安装（推荐）

  1. 下载安装包: 访问 Golang 官方网站 下载 Windows 版本的 MSI 安装包
  2. 运行安装程序: 双击下载的 .msi 文件，按照向导完成安装
    • 默认安装路径: C:\Program Files\Go
    • 安装程序会自动配置环境变量
  3. 验证安装: 打开 PowerShell 或 CMD，执行: 
    1

    go version

    如果显示版本号（如 go version go1.21.0 windows/amd64），说明安装成功

方法二：手动配置环境变量

如果安装后无法识别 go 命令，需要手动配置环境变量:

  1. 打开”控制面板” → “系统和安全” → “系统” → “高级系统设置”
  2. 点击”环境变量”按钮
  3. 在”系统变量”中找到 Path，点击”编辑”
  4. 添加 Go 的 bin 目录路径，例如: C:\Program Files\Go\bin
  5. 点击”确定”保存，重新打开终端窗口

注意: 配置环境变量后，需要重启终端才能生效

Go 基础语法要点

1. 语句分隔符

Go 语言不需要在每行末尾加分号 ;，编译器会自动添加。但以下情况需要分号:

1
2
3
4

7

// 一行多条语句时需要分号
x := 1; y := 2; z := 3

// for 循环的三个表达式之间需要分号
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

3. 标识符命名规则

  • 必须以字母或下划线开头，不能以数字开头
  • 区分大小写: name 和 Name 是不同的标识符
  • 不能使用关键字: 见下方关键字列表
  • 可见性规则:
    • 首字母大写: 公开（Public），可被其他包访问
    • 首字母小写: 私有（Private），仅当前包内可见

Go 学习路线与知识体系

基础知识体系

第一阶段：语言基础

  1. 数据类型: 整型、浮点型、布尔型、字符串、复数
  2. 变量与常量: 声明、初始化、作用域、iota 枚举
  3. 运算符: 算术、关系、逻辑、位运算
  4. 控制流程: if-else、switch-case、for 循环、break/continue
  5. 函数: 参数传递、返回值、可变参数、匿名函数、闭包
  6. 数组与切片: 声明、初始化、切片操作、扩容机制
  7. Map: 创建、增删改查、遍历、并发安全
  8. 结构体: 定义、方法、继承、JSON 序列化
  9. 指针: 基本概念、指针运算、内存管理
  10. 接口: 定义、实现、类型断言、空接口

第二阶段：高级特性

  1. 错误处理: error 接口、panic/recover、自定义错误
  2. 并发编程: goroutine、channel、sync 包、context
  3. 反射: reflect 包、类型检查、动态调用
  4. 测试: 单元测试、基准测试、表格驱动测试
  5. 标准库: io、net/http、encoding/json、time、os

第三阶段：工程实践

  1. Go Modules: 依赖管理、版本控制、私有仓库
  2. 项目结构: 标准布局、分层架构、设计模式
  3. 性能优化: 内存分配、GC 调优、profile 分析
  4. Web 开发: HTTP 服务器、路由、中间件、RESTful API
  5. 数据库: SQL、ORM、连接池、事务处理

Go 语言核心特性总结

1. 简洁的语法

Go 刻意保持语法简洁，只有 25 个关键字，降低了学习成本。相比其他语言：

  • 没有类（class）、继承、泛型（Go 1.18 前）
  • 没有异常处理（try-catch），使用 error 返回值
  • 没有复杂的元编程能力

2. 强大的并发模型

Go 的并发基于 CSP（Communicating Sequential Processes）理论：

1
2
3
4
5
6
7
8
9

// 传统多线程（复杂、易出错）
// Java/C++: Thread, Lock, Mutex, Condition...

// Go 的并发（简洁、安全）
ch := make(chan int)
go func() {
    ch <- compute()
}()
result := <-ch

3. 自动内存管理

  • 垃圾回收（GC）: 自动回收不再使用的内存
  • 逃逸分析: 编译器决定变量分配在栈还是堆
  • 内存分配器: tcmalloc 改进版，高效分配小对象

4. 快速的编译速度

  • 编译速度接近解释型语言
  • 依赖分析优化，只编译变更的部分
  • 适合快速迭代开发

5. 丰富的标准库

Go 提供了”电池included”的标准库：

  • net/http: 完整的 HTTP 客户端和服务器
  • encoding/json: JSON 编解码
  • database/sql: 数据库抽象层
  • crypto: 加密算法
  • testing: 测试框架
